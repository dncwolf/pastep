# pastep

macOS のメニューバーに常駐するクリップボード履歴マネージャー。

## 機能

- テキストとファイルパスのコピー履歴を最大 20 件保持
- `Cmd+Shift+V` でフローティングウィンドウから選択してペースト
- メニューバーアイコンからも操作可能
- 再起動後も履歴を維持（UserDefaults）

## 動作環境

- macOS 13 Ventura 以降
- Apple Silicon（arm64）

## ビルドと実行

```sh
# .app バンドルを作成して起動
make run

# バンドルのみ作成
make app

# クリーンアップ
make clean
```

## 使い方

1. `make run` で起動するとメニューバーにアイコンが表示される
2. `Cmd+Shift+V` または アイコンクリックで履歴一覧を表示
3. エントリをクリックするとクリップボードにセットされ、前のアプリへ自動ペースト

## スコープ外

画像、検索、iCloud 同期、スニペット登録には対応していない。
